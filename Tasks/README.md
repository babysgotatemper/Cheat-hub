# Tasks Directory

Папка для збереження всіх задач проекту в MD форматі.

## Структура файлів:
- `task-NNN-name.md` — опис задачі з планом виконання
- `task-NNN-name.done.md` — результати виконання

## Як це працює:
1. Коли ти даєш мені нову задачу → я автоматично створю MD-файл із планом
2. Я виконую задачу за пунктами з плану
3. Результати зберігаються в .done.md файлі
